Return type:bool

Type:int

(Read-only) The rule the vertex belongs to.

Type:Rule

left¶

(Read-only) A null descriptor if this vertex is not in the left graph, otherwise the descriptor of this vertex in the left graph.

(Read-only) A null descriptor if this vertex is not in the context graph, otherwise the descriptor of this vertex in the context graph.

(Read-only) The degree of the vertex.

Type:int

incidentEdges¶

(Read-only) A range of incident edges to this vertex.

Type:RuleIncidentEdgeRange
Raises:LogicError if it is a null descriptor.
get2DX(withHydrogens=True)¶
Returns:the x-coordinate in a 2D depiction of the rule. Different sets of coordinates exists for rendering with and wihout certain hydrogens.
Return type:float
Raises:LogicError if it is a null descriptor, or if withHydrogens is True and the vertex is a “clean” hydrogen.
get2DY(withHydrogens=True)¶
Returns:the y-coordinate in a 2D depiction of the rule. Different sets of coordinates exists for rendering with and wihout certain hydrogens.
Return type:float
Raises:LogicError if it is a null descriptor, or if withHydrogens is True and the vertex is a “clean” hydrogen.
